What to Include in Your Message

For Bug Reports:
  • Chrome version (chrome://version/)
  • Extension version (check Extensions page)
  • What you were doing when the bug occurred
  • Any error messages (check browser console: F12)
  • Screenshots if applicable
For Support Requests:
  • Clear description of the issue
  • What you've already tried
  • Which eBay site you're using (ebay.com, ebay.co.uk, etc.)
  • Browser console errors (F12 → Console tab)
For Feature Requests:
  • Description of the feature
  • Why it would be useful
  • How you envision it working
  • Any examples from other extensions

Frequently Asked Questions

Q: How long until I get a response?

We aim to respond to all inquiries within 24-48 hours. Complex technical issues may take longer to investigate.

Q: Do you offer phone support?

Currently, we only offer email support. This allows us to better track issues and provide detailed responses.

Q: Can you help with eBay account issues?

No, we can only help with issues related to the SoldinSight extension. For eBay account issues, please contact eBay directly.

Q: Is my data safe when I contact you?

Yes! We never share your information with third parties. See our Privacy Policy for details.

⚠️ Important: We are not affiliated with eBay. For eBay-specific issues (account problems, selling fees disputes, listing issues), please contact eBay support directly.

Before You Contact Us

Please try these troubleshooting steps first:

  • Refresh the page - Many issues are resolved by refreshing the eBay listing page
  • Clear cache - Open browser console (F12) and type: CacheManager.clearAll()
  • Check browser console - Press F12 and look for error messages in the Console tab
  • Reinstall extension - Remove and reinstall from Chrome Web Store
  • Check eBay status - Verify eBay.com is working properly
  • Try different listing - Test on another eBay item to see if it's item-specific
